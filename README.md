A simple, interactive penalty shootout game built with vanilla HTML, CSS, and JavaScript.

## Features

- **Interactive Gameplay**: Click directional buttons to shoot the ball
- **Smart Goalkeeper**: AI goalkeeper tries to save your shots
- **Score Tracking**: Keep track of goals scored vs saves made
- **Smooth Animations**: Visual feedback with ball and goalkeeper movements
- **Responsive Design**: Works on desktop and mobile devices
- **Modern UI**: Clean, gradient-based design with smooth transitions

## How to Play

1. Open `index.html` in your web browser
2. Click on one of the directional arrows to choose where to shoot
3. The goalkeeper will randomly dive to try to save your shot
4. Score as many goals as you can!
5. Click "Reset Game" to start over

## Running the Game

Simply open `index.html` in any modern web browser. No build process or dependencies required!

```bash

open index.html

# Option 2: Use a local server (optional)
python3 -m http.server 8000
# Then visit http://localhost:8000
```
